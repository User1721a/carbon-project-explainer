# Carbon Project Explainer Tool

A lightweight AI tool for explaining carbon project concepts, MRV logic, and sustainability frameworks in structured, accessible language.

---

## Why this project

This project was built to demonstrate:

- Understanding of carbon markets and climate terminology  
- Ability to structure complex information clearly  
- Basic API integration and deployment skills  
- Practical application of sustainability-related knowledge  

---

## Features

- Structured explanations:
  1. Short definition  
  2. Why it matters  
  3. Example  
  4. Key risks or credibility concerns  

- Example-based interaction (quick prompts)

- Simple estimation logic for real-world cases  
  (e.g. vehicle emissions → carbon credits)

---

## Focus Areas

- Carbon credits  
- MRV (Measurement, Reporting, Verification)  
- AFOLU  
- Climate project logic  
- Emissions vs removal  

---

## Tech Stack

- Next.js (App Router)
- OpenAI API
- Vercel (deployment)
